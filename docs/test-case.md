## TC-UI-001. Успешный логин

| Поле | Значение |
| --- | --- |
| ID | TC-UI-001 |
| Название | Успешный логин пользователя |
| Предусловия | Пользователь находится на странице логина SauceDemo |
| Тестовые данные | `standard_user` / `secret_sauce` |
| Шаги | 1. Открыть сайт SauceDemo. <br> 2. Ввести валидный логин. <br> 3. Ввести валидный пароль. <br> 4. Нажать кнопку Login. |
| Ожидаемый результат | Пользователь успешно авторизуется и попадает на страницу товаров `/inventory.html`. |
| Статус | Automated |
| Результат | Passed |
